# Security Policy

## Purpose

vuln-spa is an intentionally vulnerable application designed for security training and benchmarking.

The vulnerabilities included in this repository are expected behavior and should not be reported as security issues.

## Reporting Security Issues

Please report only:

- vulnerabilities affecting the benchmark infrastructure
- CI/CD pipeline security issues
- dependency vulnerabilities not intentionally introduced
- supply chain risks

## Contact

Open a GitHub Security Advisory or GitHub Issue.

## Scope

Out of scope:

- OWASP Top 10 vulnerabilities intentionally present in the application
- insecure coding examples documented in the repository
- attack paths described in benchmark scenarios
