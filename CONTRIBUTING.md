# Contributing

Contributions are welcome.

## Ways to contribute

- add new vulnerability scenarios
- improve documentation
- add DevSecOps integrations
- improve benchmark automation
- add scanner result mappings

## Pull Requests

1. Fork repository
2. Create feature branch
3. Commit changes
4. Open Pull Request

## Coding Standards

- Java 21
- Spring Boot 3
- React frontend
- Security-focused examples
